# 📂 7th semenester

## 📚 Overview
This repository contains two main lab projects:
1. **Microprocessors Lab** - Focused on embedded systems using **ATMega328PB**.
2. **OS Lab** - Covers **Linux kernel development** and **file system implementation**.

## 🛠️ Microprocessors Lab
- **Topics Covered:** Timers, ADC, LCD displays, TWI, expansion ports, One-Wire, and UART.
- **Key Project:** A **hospital IoT system** integrating multiple sensors and WiFi connectivity using **ESP8266**.
- **Technologies:** C, Assembly, Embedded Systems, Sensor Networks.

## 💻 OS Lab
- **Lunix Driver:** A **character device driver** for a wireless sensor network in **Linux**.
- **Ext2-Lite:** A lightweight **Ext2 file system** implementation.
- **Technologies:** Linux Kernel, Device Drivers, File Systems, Synchronization, QEMU/KVM.

## 📂 Repository Structure
```
/
├── MicroLab/   # Microprocessors Lab exercises
└── OS_Lab/     # Operating Systems Lab exercises
```

## 🛠️ Tools & Environments
- **MicroLab:** Custom ATMega328PB PCB, AVR-GCC.
- **OS Lab:** Linux Kernel 6.11, QEMU/KVM, GCC.
